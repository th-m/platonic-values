# Philosophical Decision Tree — Session Notes

## Goal
Build a binary yes/no decision tree walking through fundamental beliefs, starting from truth and branching through agency, morality, and meaning.

## Core Structure

### Starting Question: Does objective truth exist?
- **Yes** → Realism branch
- **No** → Truth theory branch

---

## Truth Theory Branch

### Binary Decision Path
1. Does truth require matching something outside the mind?
   - Yes → **Correspondence Theory**
   - No → Q2
2. Is "true" a substantive, meaningful predicate?
   - Yes → Q3
   - No → Q4
3. Is a belief true because it coheres with other beliefs?
   - Yes → **Coherentism**
   - No → **Pragmatism**
4. Are there any truths at all?
   - Yes (but "true" is linguistic shorthand) → **Deflationism**
   - No → **Nihilism**

### Plain Language Framings
| Theory | Layman Summary | Core Intuition |
|---|---|---|
| Correspondence | A map must match the territory | Truth is out there, independent of us |
| Coherentism | A story that hangs together without contradictions | Truth is a web, not a mirror |
| Pragmatism | If it ain't broke don't fix it | Truth is what works |
| Deflationism | "True" is just emphasis, not a deep concept | Truth is a linguistic convenience |
| Nihilism | Nothing is real, nothing counts | Truth is an illusion |

### Relationship to Reality Pattern
- Correspondence → *submission* to reality
- Coherentism → *construction* of reality
- Pragmatism → *negotiation* with reality
- Deflationism → *indifference* to reality
- Nihilism → *rejection* of reality

---

## No-Truth Sub-branches

### Pragmatism Fork
- True for whom?
  - Individual → **Subjectivism**
  - Community → **Social Constructivism** (Rorty, Kuhn)
- Can some beliefs be better than others?
  - No → collapses into Relativism

### Nihilism Fork
- Does anything matter at all?
  - No → **Passive Nihilism** (Schopenhauer)
  - Yes, I create it → **Active Nihilism / Existentialism**

### Relativism Fork
- Are all frameworks equally valid?
  - Yes → **Full Relativism** (self-defeating)
  - No → **Perspectivism** (Nietzsche)

---

## Realism Branch

### Is reality mind-independent?
- Yes → **Realism** → determinism questions
- No → **Idealism** → whose mind?

### Idealism Forks
| Starting Point | Branch | Key Thinkers |
|---|---|---|
| One infinite mind | Absolute Idealism | Hegel, Bradley |
| My mind alone | Solipsism | — |
| Many minds + shared structure | Intersubjective Idealism | Husserl |
| Structured by mind, not made of it | Kantian / Transcendental Idealism | Kant |

#### Absolute Idealism: Is reality progressing?
- Yes, rationally → Hegelian Dialectic
- Yes, spiritually → Theistic Idealism
- No → Static Monism (Parmenides)

#### Kantian: Can we know the thing-in-itself?
- No → Stay with Kant
- Yes, partially → Critical Realism
- Meaningless question → Logical Positivism

#### Intersubjective: What grounds shared structure?
- Language → Linguistic Idealism (Wittgenstein)
- Social practice → Social Constructivism (Berger & Luckmann)
- Embodied experience → Phenomenology (Merleau-Ponty)

---

## Realism Downstream

### Is everything causally determined?
- Yes → **Hard Determinism**
- No → Does genuine agency exist?
  - Yes → **Libertarian Free Will / Compatibilism**
  - No → **Epiphenomenalism / Illusionism**

### Does objective morality exist?
- Yes → **Moral Realism**
- No → **Relativism / Error Theory / Expressivism**

### Does intrinsic meaning exist?
- Yes → **Theism / Platonism / Natural Law**
- No → **Existentialism / Absurdism / Nihilism**

---

## Notes & Observations
- Nihilism and Existentialism share the same premise but diverge entirely on agency
- Social Constructivism appears in both the pragmatist and intersubjective idealism branches — accurately
- "Is consciousness fundamental or derived?" is a strong alternative starting question that reorganizes the tree around mind rather than truth
- Each truth theory mirrors a pattern in how people relate to values generally — worth integrating into therapeutic applications
