# Book Architecture (Framework-First, Nature-Literary Voice)

## Editorial Intent

This architecture treats the manuscript as a field guide to moral ecology:

- Define the map early and clearly.
- Move through the five value habitats with sensory, natural-history openings.
- Build many smaller emotional peaks by ending each unit with a lived human turn.
- Keep theology available as comparative evidence, not a required premise.

## Part-Level Intent and Flow

### Part I: Reading the Terrain

- Purpose: establish the [[value-framework|5x3 framework]] as a usable map, not an abstract puzzle.
- Output: readers can identify domain, dimension, and mean/pole location in ordinary life.

### Part II: The Five Habitats

- Purpose: walk domain-by-domain through [[five-fundamental-values|virtue-commodities]] using natural-history scenes.
- Output: each value domain feels lived, embodied, and narratively memorable.

### Part III: Distortion and Metamorphosis

- Purpose: show how value disorders arise and how transformation happens.
- Required integrations: deficit/excess, omission/commission, exploitation vs appreciation, and [[ideation/psychology/inverted-hierarchy|inverted hierarchy]].
- Output: readers can diagnose distortion without reducing persons to pathology.

### Part IV: Stewardship and Return

- Purpose: move from theory to ethics, institutions, and care as the culminating expression.
- Output: practical commitments for individuals and communities.

## Editorial Defaults (Locked)

- Preserve core vocabulary: Platonic value theory, virtue-commodities, 5x3 framework.
- Keep core argumentative lane mostly secular.
- Use theological material as comparative and optional support.
- Preserve existing note graph; no automatic moves, merges, or renames.

## Acceptance Tests

1. Clarity: each unit has one central claim a thoughtful general reader can restate after one sitting.
2. Motif Growth: each recurrence adds meaning rather than repeating symbol decoration.
3. Emotional Cadence: one meaningful emotional turn per unit; emotional function rotates across units.
4. Lexical Consistency: key terms stay definition-stable across manuscript and glossary.
5. Audience Access: argument remains coherent without requiring theological assent.
