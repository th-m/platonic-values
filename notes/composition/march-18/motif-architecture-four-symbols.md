# Motif Architecture: Four Symbols, Four Jobs

## Purpose

Formalize a repeatable motif system so each recurrence deepens argument and emotional force.

## The Four Motifs

### Butterfly

Conceptual function: identity persistence across radical transformation.

Emotional function: tenderness, grief-transformed, earned hope.

Primary framework links: [[five-fundamental-values]], [[ideation/psychology/inverted-hierarchy]], [[composition/butterfly-motif]].

### Riverbanks / Constraint

Conceptual function: limits create capacity; boundaries make flow and emergence possible.

Emotional function: sobriety, grounding, responsibility.

Primary framework links: [[value-framework]], [[ideation/metalogic/set-theory-meta-math]].

### Migration Path

Conceptual function: optimal route through time; direction matters more than speed.

Emotional function: orientation, perseverance, practical agency.

Primary framework links: [[value-framework]], [[decision-tree]].

### Night Sky / Infinity

Conceptual function: personhood exceeds reduction to utility or mechanism.

Emotional function: awe, humility, moral regard.

Primary framework links: [[ideation/metalogic/set-theory-meta-math]], [[ideation/philosophy/ontological-foundations]].

## Recurrence Rules

1. Each chapter gets one primary motif beat and one light echo.
2. Primary beat must perform conceptual work, not decorative atmosphere.
3. No repeated emotional function in back-to-back chapters.
4. Recurrence arc per motif: scene -> interpretation -> integration -> ethical demand.
5. Theology can appear in motif interpretation, but secular framing remains sufficient.

## Emotional Cadence Guardrail

Use a rotating emotional sequence to avoid monotone emphasis:

Wonder -> Grounding -> Clarity -> Vulnerability -> Kinship -> Recognition -> Awe -> Reverence -> Alarm -> Accountability -> Reframing -> Responsibility -> Tenderness -> Quiet resolve.
