# Glossary Architecture and Starter Lexicon

## Purpose

Create a three-layer glossary system that keeps language accessible for general readers while preserving full conceptual precision.

## Three-Layer Architecture

### Layer 1: Reader Glossary

Audience: thoughtful general readers.

Use: plain-language definitions in back matter (20-30 terms).

Goal: remove friction without requiring technical training.

### Layer 2: Technical Lexicon

Audience: close readers, researchers, advanced discussion groups.

Use: exact matrix logic, distinctions, and boundary conditions.

Goal: definition stability across manuscript, talks, and derivative writing.

### Layer 3: Motif Lexicon

Audience: readers tracking narrative structure and symbolic argument.

Use: symbol -> conceptual function -> emotional function.

Goal: ensure motif recurrence adds meaning and not decorative repetition.

## Standard Entry Format (All Layers)

For each glossary entry, include all fields:

1. One-line definition
2. Technical meaning
3. Common confusion
4. Related framework terms
5. Where it appears

## Starter Terms (Locked)

### virtue-commodity

One-line definition: One of the five irreducible domains of human value.

Technical meaning: A foundational value domain in Platonic value theory: Physical, Others, Self, Mental, Spirit.

Common confusion: Treating virtue-commodities as personality types rather than value domains.

Related framework terms: [[five-fundamental-values]], [[value-framework]], domain.

Where it appears: Part I framing and all Part II habitat chapters in [[composition/march-18/book-architecture-framework-first]].

### 5x3 framework

One-line definition: A matrix mapping five value domains across three dimensions.

Technical meaning: The full value map: five domains x internal/spatial/temporal dimensions, each with deficit-mean-excess spectra.

Common confusion: Assuming the framework is only descriptive and not also diagnostic.

Related framework terms: [[value-framework]], domain, dimension, deficit, mean, excess.

Where it appears: Part I, Ch. 2 and throughout Parts II-IV.

### domain

One-line definition: A major territory of value.

Technical meaning: One full row of the 5x3 matrix (Physical, Others, Self, Mental, Spirit).

Common confusion: Using domain and dimension interchangeably.

Related framework terms: [[five-fundamental-values]], dimension, virtue-commodity.

Where it appears: Part I orientation and Part II habitat sequence.

### dimension

One-line definition: The mode in which a value domain is expressed.

Technical meaning: Internal, spatial, and temporal expression lanes across each domain.

Common confusion: Treating dimensions as a hierarchy instead of parallel analytical lenses.

Related framework terms: [[value-framework]], domain, omission, commission.

Where it appears: Part I, Ch. 2 and all technical sidebars.

### deficit

One-line definition: Too little of a value.

Technical meaning: The deficiency pole of a value spectrum in a specific domain and dimension.

Common confusion: Equating deficit with weakness of character in all cases.

Related framework terms: mean, excess, value disorder.

Where it appears: Part III, Ch. 8.

### mean

One-line definition: The virtuous center between value deficiency and surplus.

Technical meaning: Context-sensitive balanced expression of a value, following Aristotelian logic.

Common confusion: Mistaking mean for mediocrity or emotional blandness.

Related framework terms: deficit, excess, [[ideation/philosophy/virtues/paradox-of-sin]].

Where it appears: Part I foundations and Part III diagnostics.

### excess

One-line definition: Too much of a value in a distorted direction.

Technical meaning: The surplus pole where value pursuit overruns adjacent values.

Common confusion: Assuming all intensity is excess.

Related framework terms: mean, deficit, value disorder.

Where it appears: Part III, Ch. 8.

### omission

One-line definition: Value violation by failing to act.

Technical meaning: Passive mode of distortion where value obligations are neglected.

Common confusion: Treating omission as morally neutral.

Related framework terms: commission, [[value-framework#Omission and Commission: The Dual Modes of Value Violation]].

Where it appears: Part III, Ch. 9.

### commission

One-line definition: Value violation by wrongful action.

Technical meaning: Active mode of distortion where behavior directly harms value alignment.

Common confusion: Assuming commission is always worse than omission in every context.

Related framework terms: omission, deficit, excess.

Where it appears: Part III, Ch. 9.

### value disorder

One-line definition: A patterned distortion in relationship to a legitimate value.

Technical meaning: Any persistent misalignment in domain, dimension, or pole expression.

Common confusion: Treating disorders as separate from values rather than distortions of values.

Related framework terms: [[value-framework]], deficit, excess, omission, commission.

Where it appears: Parts III-IV.

### exploitation

One-line definition: Using value structures instrumentally without reverence for persons or goods.

Technical meaning: Strategic extraction from value systems while refusing intrinsic regard for what is being used.

Common confusion: Confusing competence in value dynamics with moral integrity.

Related framework terms: appreciation, [[value-exploitation-vs-appreciation]], value disorder.

Where it appears: Part III, Ch. 10.

### appreciation

One-line definition: Relating to values with reverence, care, and non-instrumental regard.

Technical meaning: Participatory alignment with value goods at the mean, with respect for adjacent constraints.

Common confusion: Treating appreciation as sentiment without action.

Related framework terms: exploitation, custodianship, mean.

Where it appears: Part III, Ch. 10 and Part IV.

### inverted hierarchy

One-line definition: A model where meaning functionally organizes lower tiers.

Technical meaning: Functional-priority ordering where Spirit grounds and orders downstream value integration.

Common confusion: Confusing functional priority with developmental sequence.

Related framework terms: [[ideation/psychology/inverted-hierarchy]], functional priority, developmental sequence.

Where it appears: Part III, Ch. 10.

### functional priority

One-line definition: The order in which values organize one another in mature functioning.

Technical meaning: Normative causal-ordering claim about integration, distinct from emergence timing.

Common confusion: Assuming what comes first developmentally is always what governs adult coherence.

Related framework terms: inverted hierarchy, developmental sequence.

Where it appears: Part III and Part IV integration.

### developmental sequence

One-line definition: The order values usually appear during growth and maturation.

Technical meaning: Descriptive account of emergence across lifespan and context.

Common confusion: Mistaking development order for normative order of governance.

Related framework terms: functional priority, [[value-framework]], [[ideation/psychology/inverted-hierarchy]].

Where it appears: Part I and Part III comparison passages.

### salience

One-line definition: What stands out as meaningful, relevant, and action-guiding.

Technical meaning: Priority weighting in perception and valuation that shapes memory, judgment, and behavior.

Common confusion: Treating salience as mere intensity rather than structured significance.

Related framework terms: meaning, attention, [[composition/butterfly-motif]].

Where it appears: Prologue and Part II, Ch. 7.

### custodianship

One-line definition: Responsible care for what one does not ultimately own.

Technical meaning: Spirit/spatial value orientation toward stewardship over domination or neglect.

Common confusion: Reducing custodianship to private preference instead of ethical obligation.

Related framework terms: [[value-framework]], exploitation, appreciation, ultimacy.

Where it appears: Part II, Ch. 7 and Part IV.

### ultimacy

One-line definition: How a person orients toward final meaning and highest concern.

Technical meaning: Spirit/temporal value concerning ultimate commitments across time.

Common confusion: Equating ultimacy only with explicit religiosity.

Related framework terms: meaning, custodianship, [[value-framework]].

Where it appears: Part II, Ch. 7 and epilogue.

### metamorphosis

One-line definition: Deep transformation that preserves identity-bearing pattern.

Technical meaning: Structural reorganization across levels where continuity is pattern-based, not substrate-based.

Common confusion: Treating metamorphosis as total replacement with no continuity.

Related framework terms: butterfly, salience, continuity-through-discontinuity.

Where it appears: Prologue, Part III, and epilogue.

### continuity-through-discontinuity

One-line definition: Real continuity can survive radical outward change.

Technical meaning: Identity persistence via stable relational pattern across substrate disruption.

Common confusion: Confusing continuity with stasis.

Related framework terms: metamorphosis, salience, [[composition/butterfly-motif]], [[ideation/metalogic/set-theory-meta-math]].

Where it appears: Prologue and epilogue.

## Motif Lexicon (Layer 3, Working)

### Butterfly

Symbol: metamorphosis and preserved identity.

Conceptual function: continuity-through-discontinuity.

Emotional function: tenderness, grief transfigured into agency.

### Riverbanks / Constraint

Symbol: bounded flow.

Conceptual function: limit as enabler.

Emotional function: grounded responsibility.

### Migration Path

Symbol: route and orientation over time.

Conceptual function: optimal path through constraint.

Emotional function: perseverance and directional hope.

### Night Sky / Infinity

Symbol: scale beyond utility.

Conceptual function: personhood exceeds reduction.

Emotional function: awe and moral humility.
